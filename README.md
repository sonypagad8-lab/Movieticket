# 🎬 Movieticket

Movie Ticket Booking System is a web-based application developed using Spring Boot and Thymeleaf. The application allows users to browse movies, view show timings, select seats, and book movie tickets online easily.



## 🚀 Features

* User Registration & Login
* Browse Available Movies
* View Theater & Show Timings
* Seat Selection & Ticket Booking
* Booking Confirmation
* Admin Management
* Secure Authentication
* Responsive User Interface



## 🛠 Technologies Used

* Java
* Spring Boot
* Thymeleaf
* MySQL
* HTML
* CSS
* Bootstrap
* Maven
* JPA / Hibernate



## 📂 Modules

### User Module

* User Registration
* Login Authentication
* Profile Management

### Movie Module

* Add/View Movies
* Manage Movie Details

### Theater Module

* Manage Theaters & Screens
* Show Timings Management

### Booking Module

* Seat Selection
* Ticket Booking
* Booking History

### Admin Module

* Manage Users
* Manage Movies & Shows

---

## 🎯 Project Objective

The main objective of this project is to automate the movie ticket booking process and provide users with an easy online platform for booking movie tickets anytime and anywhere.

---

## ⚙️ Installation Steps

1. Clone the repository

```bash
git clone https://github.com/your-username/Movieticket.git
```

2. Open the project in your IDE

3. Configure MySQL database in `application.properties`

4. Run the project using Spring Boot

5. Open browser:

```bash
http://localhost:8080
```
